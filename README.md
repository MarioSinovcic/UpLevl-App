# UpLevl-App

# Requirements:
- Node.js
- npm or yarn
- MacOS with XCode
    - Homebrew
        - watchman
    - XCode
    - cocoapods
    
Make sure you follow this [guide](https://reactnative.dev/docs/environment-setup) for installing the required packages

# How to run locally:
1. Open the `UpLevl` directory in terminal
2. Run `npm i`
3. Run `npx react-native run-ios`
